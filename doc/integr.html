Вы можете бесплатно интегрировать (вставить) окно "Час ЕГЭ" в свой сайт через iframe. Примеры:

  • ege-ok.ru
  • matematikalegko.ru
  • 4ege.ru - оболочка случайного задания
  • 4ege.ru - полный тест

На данный момент рекомендуется интеграция через скрипты. Всё, что Вам нужно сделать - это добавить в нужное место Вашей страницы несложный код, например: 

	<script src="//www.math.vsu.ru/chas-ege/lib/browser.js">
	</script>
	<script>
		chas.integrate("https://www.math.vsu.ru/chas-ege/sh/polnmat.html",{
			style:{
				strCSS:"#menucenter{font-size:12px;} #inf{height:110px;}",
				pan:'lightblue',
				fon:'#fff',
				menuLinkTarget:'_self',
			},
		},"98%",1500);
	</script>

Пример выполнения такого кода приведён ниже. Поясним теперь значение каждого элемента кода.

  • Первый тэг script подключает собственно скрипт интеграции.
  • Второй тэг script содержит вызов единственной функции chas.integrate(); Всё её параметры являются необязательными (в понимании javascript, т. е. можно не указывать параметры, начиная с некоторого).
    1. Первый параметр - адрес интегрируемой страницы. Рекомендется использовать случайное задание https://www.math.vsu.ru/chas-ege/sh/sluch.html или полный тест https://www.math.vsu.ru/chas-ege/sh/polnmat.html
    2. Второй параметр - настройки. Пока доступен только один подпараметр - настройки оформления style. Могут иметь следующие необязательные свойства (в понимании JSON, т. е. можно опускать любые):
      • strCSS - самый мощный параметр настройки. Строка со стилями, которая будет записана в тэг style внутри страницы тренажёра. Может содержать любые CSS-выражения, исключая символы < и >
      • pan и fon - цвета панелей и фона соотвественно. Указываются как в HTML.
      • menuLinkTarget - значение атрибута target у ссылок в меню. Для того, чтобы всё, что возможно, открывалось внутри iframe, установить в значение '_self'.
    3. Третий и четвёртый параметры - ширина и высота формируемого iframe. По умолчанию 900 и 1500 пикселей соответственно.
